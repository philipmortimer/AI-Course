This is a Chess program that includes a chess AI. To play simply log in as a guest and go to offline / local games. 
This system works with Windows, however it may not full fully function with other operating systems. To run the game, open it in an IDE
like NetBeans or IntelliJ and run it with an appropraite version of Java installed. Alterantively, run the ".jar" at the highest directory level.
Either way, you will need to have Java installed. Please follow this tutorial to do so: 
https://github.com/philipmortimer/GeneralCourse-Materials-AI-Course/blob/main/AI%20Intro%20Course%20Install%20Guide%20-%20Java.pdf
